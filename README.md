## How to track

Checkout the files like so:

```sh
git clone --branch slightly-blue-team git@github.com:dekhtyar/CSC366-Spring2013.git
cd CSC366
```

Do some work

Commit your work

```sh
git add new-file modified-file
git commit -m 'made some changes'
```

Push to our ref

```sh
git push
```
